## Academic Projects

1. [Gas Laws Desktop Application, 2019](https://github.com/cindyddang/GasLawApplication)
My first project that associates with GUI. It calculates the missing properties from input temperature, volume or/and pressure by the formula of Boyle's Law, Charles Law, Gay-Lussac's Law or Combined Gas Law.

2. [Summer Spending, 2019](https://github.com/cindyddang/SummerSpending)
A summer project that uses SQL to create database and store data. It simply allows user to add/edit/delete data.

3. [Financial Assistance Desktop Application, 2020-2021](https://github.com/cindyddang/FinancialAssistProgram)
An Internal Assessment project for my IB Computer Science High Level class. This program was built using SQL and Java. Its function is to help with calculate and keep track of finance within a household. It analyzes spendings that associate with multiple people, calculate and suggest who should pay who for how much every month.
